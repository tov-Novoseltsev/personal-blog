Hosting on Github

 24 May 2015

I knew it was possible, but I din't know it would be that easy. So, to host your static html content on Github you need:

  • A Github account;
  • Create new repository;
  • Open "gh-pages" branch and push there your static web content.

Want a custom domain name?

  • Go buy one;
  • Configure an A record to point to github ip addresses;
  • Add a file named "CNAME" to repository and put your custom domain name in it.
